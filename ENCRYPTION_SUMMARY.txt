⚡ DÉMARRAGE RAPIDE - 3 COMMANDES
═══════════════════════════════════════════════════════

1. GÉNÉRER LA CLÉ (première fois uniquement)
   ▼ Windows GUI:      Double-cliquer encrypt_init.bat
   ▼ Windows CMD:      encrypt_init.bat
   ▼ PowerShell:       .\encrypt_sources.ps1 -Action init
   ▼ PHP CLI:          php bin/encrypt.php init
   
   Résultat: Clé générée et sauvegardée en .env

───────────────────────────────────────────────────────

2. CHIFFRER LES FICHIERS
   ▼ Windows GUI:      Double-cliquer encrypt_all.bat
   ▼ Windows CMD:      encrypt_all.bat
   ▼ PowerShell:       .\encrypt_sources.ps1 -Action encrypt
   ▼ PHP CLI:          php bin/encrypt.php encrypt
   
  Résultat: versions .encrypted créées pour les cibles par défaut

───────────────────────────────────────────────────────

3. VÉRIFIER L'ÉTAT
   ▼ Windows GUI:      Double-cliquer encrypt_status.bat
   ▼ Windows CMD:      encrypt_status.bat
   ▼ PowerShell:       .\encrypt_sources.ps1 -Action status
   ▼ PHP CLI:          php bin/encrypt.php status
   
   Résultat: Affiche statistiques de chiffrement

✨ CARACTÉRISTIQUES CLÉS
═══════════════════════════════════════════════════════

CRYPTOGRAPHIE
  • Algorithme: AES-256-CBC (NIST FIPS 197)
  • IV Aléatoire: OpenSSL random_pseudo_bytes()
  • Encodage: Base64
  • Sécurité: Militaire

AUTOMATISATION
  • Détection auto de .encrypted
  • Déchiffrement au runtime (transparent)
  • Zéro modification applicative
  • Support multi-interface (CLI, PS, Batch)

PERFORMANCE
  • Overhead: < 5ms par démarrage
  • Déchiffrement en mémoire (pas I/O réseau)
  • Pas de dépendance externe (OpenSSL inclus PHP)

MAINTENABILITÉ
  • Configuration centralisée (encryption.php)
  • Loader réutilisable (encrypted_loader.php)
  • Scripts d'automatisation (bin/encrypt.php)
  • Documentation complète (ENCRYPTION.md)
