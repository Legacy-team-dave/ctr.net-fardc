QUICK START - CTR.NET-FARDC

1) Lancer Laragon
2) Ouvrir le projet dans C:\laragon\www\ctr.net-fardc
3) Initialiser le chiffrement (v1.1.0+):
   - Double-cliquer: encrypt_init.bat
   - Ou en CLI: php bin/encrypt.php init
4) Ouvrir : http://127.0.0.1/ctr.net-fardc/splash_screen.php

Scripts de lancement disponibles:
- START.bat
- INSTALLER.bat
- launch.bat
- launch.ps1
- lanceur.ps1

Outils de chiffrement:
- encrypt_init.bat       (Initialiser la clé secrète)
- encrypt_all.bat        (Chiffrer tous les fichiers sensibles)
- encrypt_status.bat     (Vérifier l'état du chiffrement)
- encrypt_list.bat       (Lister les fichiers chiffrables)

API REST mobile (v1.2.0+):
- api/auth.php           (Authentification token Bearer)
- api/controles.php      (Recherche + validation contrôle)
- api/profil.php         (Profil utilisateur)
- api/controles_poll.php (Polling temps réel contrôles mobiles)

Sauvegarde automatique incrémentale (8h):
- Exécuter: setup_backup_task.bat
- Exécution autonome sans visiter les pages
- Archive ZIP avec CSV + XLSX des nouveaux éléments uniquement
- Purge auto: supprimer les archives de plus de 60 jours
- Purge auto: conserver seulement 30 dernières archives non identiques
- Purge manuelle: powershell -File run_backup_purge.ps1 -MaxKeep 30

Nettoyage automatique des caches (v1.5.0+):
- Exécution automatique: intégré au job planifié (toutes les 8h)
- Cibles: fichiers temp XLSX, verrous, tokens expirés, logs > 90 jours
- Nettoyage manuel: run_cache_cleanup.bat 90
