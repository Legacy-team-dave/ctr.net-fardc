## 🎯 Points Clés de la Structure

### Séparation des Profils
```
ADMIN ONLY (ADMIN_IG):
  └── modules/administration/*       ✅ Protégé par check_profil(['ADMIN_IG'])

OPERATEUR:
  ├── modules/militaires/*           ✅ Accès opérateur (ADMIN_IG aussi)
  ├── modules/controles/*            ✅ Accès opérateur (ADMIN_IG aussi)
  └── modules/rapports/*             ✅ Accès opérateur (ADMIN_IG aussi)

CONTROLEUR (interface mobile top-nav):
  └── modules/controles/ajouter.php ✅ Saisie du contrôle uniquement
```

### Architecture MVC
```
Model (Entités) :
  └── app/classes/
      ├── User.php
      ├── Militaire.php
      ├── Controle.php

View (Templates) :
  ├── includes/header.php
  ├── includes/footer.php
  ├── modules/*/page.php

Controller (Logique) :
  ├── app/controllers/
  │   ├── AdminController.php        (Admin)
  │   └── UserController.php         (Users)
  └── api/*.php                      (AJAX)

Service (Métier) :
  └── app/services/
      ├── UserService.php
      ├── MilitaireService.php
      ├── ControleService.php
```

### Sécurité (Middleware)
```
Tous les modules incluent:
  └── RoleMiddleware::requireRole()
      ou RoleMiddleware::requireAdmin()
      ou RoleMiddleware::requireLogin()
```

### Styles (Barlow)
```
assets/css/styles.css
  ├── @import url('Barlow Google Fonts')
  ├── body { font-family: 'Barlow', sans-serif }
  └── @import url('custom.css')

Tous les pages incluent:
  └── <link rel="stylesheet" href="/assets/css/styles.css">
```
